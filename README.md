# 🌏 AsePal-Web

> 面向东南亚留学生的多语言 AI 交互平台  
> 基于 React + Vite + TypeScript 构建

---

## 📌 项目简介

**AsePal-Web** 是一个基于现代前端技术栈构建的多语言 AI 交互平台，  
致力于为东南亚来华留学生提供跨文化沟通支持与智能辅助服务。

本项目不仅是一个 AI 对话系统，同时也是一个结构清晰、可扩展的前端架构示例项目。

---

## ✨ 核心功能

### 💬 AI 实时对话系统

- 支持流式响应（SSE）
- 会话管理机制
- 气泡式聊天界面
- 自动滚动到底部
- 可扩展语音交互能力（未来计划）

---

### 🌐 多语言支持（i18n）

采用模块化国际化设计，支持多语言切换。

当前支持语言：

- 🇨🇳 简体中文
- 🇺🇸 英语
- 🇲🇲 缅甸语
- 🇹🇭 泰语
- 🇻🇳 越南语

特点：

- 按功能模块拆分翻译文件
- 可扩展语言结构
- 类型安全

---

### 🔐 认证系统

- 用户登录
- 用户注册
- 找回密码
- 基于 Token 的身份验证

---

### 📬 投诉与反馈系统

- 表单提交
- 支持附件上传
- 投诉分类管理
- 面向国际学生问题收集

---

### 📄 法律与信息页面

- 隐私政策
- 使用条款
- 关于我们

---

## 🏗 项目架构设计

本项目采用 **Feature-Based（功能模块化）架构设计**。

### 🧠 架构设计原则

- 高内聚、低耦合
- 模块独立导出
- 类型集中管理
- 便于扩展与维护
- 清晰的职责划分

---

## 🛠 技术栈

| 技术           | 说明         |
| -------------- | ------------ |
| React 19       | 前端 UI 框架 |
| Vite 7         | 构建工具     |
| TypeScript 5   | 类型系统     |
| React Router 7 | 路由管理     |
| Tailwind CSS 4 | 样式框架     |
| i18next        | 国际化方案   |
| Axios          | 网络请求     |

## 🚀 本地运行指南

### 1️⃣ 克隆项目

```bash
git clone https://github.com/你的用户名/SinoSEA-Vite.git
cd SinoSEA-Vite
2️⃣ 安装依赖
pnpm install
# 或
npm install
3️⃣ 配置环境变量

在项目根目录创建 .env 文件：

VITE_API_BASE_URL=https://your-api-domain.com
VITE_APP_ENV=development
可参考 .env.example 文件。

4️⃣ 启动开发环境
pnpm dev

默认访问地址：

http://localhost:5173
📦 构建生产环境
pnpm build

构建输出目录：

dist/
```
